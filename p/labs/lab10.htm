Container Classes

Container Classes: classes that maintain collections of objects.

Example:

class Person_Container
{
  public:
  ... ...
  private:
    Person* array;
  ... ...
}


The operations on container classes can be:

member functions of the container class.
global functions that take a container argument.

Three concepts of container classes:

the kind of container (list-based, array-based)
the kind of objects stored in the container (Person, int)
the kind of operations on the elements stored in the container ("do something for each element")

Usually, there is a strong coupling between the three concepts.
 

Generic Programming

Generic Programming: programming with types as parameters.

Generic programming is a method through which we can remove (or strongly reduce) the strong coupling between containers, container elements and operations on the elements of the container.

 

Function Templates

C++ supports generic programming through the template mechanism.

Definition:

template<typename T>
T max(const T& a, const T& b)
{
  ... ...
}

The template with formal argument T will be instantiated by the compiler with the actual argument, e.g. int.

Note:

There is NO automatic type conversion for template arguments. E.g. cout << max(4, 5.5) // Error!
You can help by explicitly instantiating the function template. E.g. cout << max<double>(4, 5.5)
Another way is to use more than one formal arguments. E.g. template <typename T1, typename T2>

Please see a small example of function template.
 

Class Templates
Class template: a prescription for creating a class in which one or more types or values are parameterized.

Definition:

template <class T>
class Queue
{
  public:
    Queue();
    ~Queue();
    T& remove();
    void add( const T&)
  private:
    ... ...
};

Usages of class template:

Queue<int> qa;
Queue<string> qb;

The template keyword tells the compiler that the class definition that follows will manipulate one or more unspecified types. At the time the actual class code is generated from the template, those types must be specified so that the compiler can substitute them.
